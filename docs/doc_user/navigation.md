# Outils de navigation

Pour naviguer dans la carte, l'utilisateur à la possibilité d'utiliser
trois outils présents sur la droite de l'interface :

-   Zoom + ( ![Zoom In](../_images/user/navigation/zoom_in.png) )

-   Zoom sur la position initiale de la carte (
    ![Zoom full extent](../_images/user/navigation/zoom_full_extent.png)
    )

-   Zoom - ( ![Zoom out](../_images/user/navigation/zoom_out.png) )

    > width  
    > 16 pt
    >
    > width  
    > 16 pt
    >
    > width  
    > 16 pt

À noter qu'il est également possible de naviguer à l'aide de la souris
et du clavier :

-   Avec la roulette : Zoom + / - ,
-   Double clic-gauche : Zoom +
-   Maj + Double clic-gauche : Zoom -
-   Clic-gauche : déplacement de la carte, sans changer le niveau de
    zoom.

Pour modifier la position de départ du zoom (et donc celle appliquée
avec l'outil
![Zoom full extent](../_images/user/navigation/zoom_full_extent.png) ),
veuillez vous référer à la page [Configurer - les options de la carte](../doc_tech/config_map.md).
