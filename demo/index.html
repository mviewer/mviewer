Contexte

Utiliser un contexte de carte WMC

Cet exemple illustre l'utilisation d'un contexte de carte servant de source au panel des thématiques. Les fonds de cartes et les outils sont ceux de la configuration par défaut. Il est possible d'appeler plusieurs wmc en utilisant la virgule comme séparateur.

Facilel93proxy

No proxy

Cet exemple illustre l'utilisation du mviewer sans proxy AJAX

Pour plus de souplesse, l'utilisation d'un proxy AJAX est recommandée. Toutefois, il est possible d'exploiter des flux locaux ou des flux externes (si CORS est activé) - c'est le cas de la BAN et de GéoBretagne.

Proxy

Cet exemple illustre l'utilisation du mviewer avec proxy AJAX

un proxy cgi peut être utilisé. Ce dernier sert d'intermédiaire pour requêter des serveurs distants. Ce proxy doit être installé sur le même domaine/machine que le mviewer. Plus de détail ici : proxy.

FacilebanLocalisation

Base Adresse Nationale

Se localiser avec la BAN

Il est possible d'utiliser l'API search de adresse.data.gouv.fr afin de se localiser à partir d'une recherche plein texte saisie dans la zone de recherche. Ce service peut être utilisé sans proxy AJAX et sans clé.

Sources Live!

FacilegeoportailopenlsLocalisation

OpenLS

Se localiser avec le GéoPortail

Il est possible d'utiliser l'API D'autocompletion du GéoPortail afin de se localiser à partir d'une recherche plein texte saisie dans la zone de recherche. Ce service peut être utilisé sans proxy AJAX mais nécessite l'utilisation d'un clé IGN.
